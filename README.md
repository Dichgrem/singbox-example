## singbox-example

Sing-box 是一个现代、高性能、功能强大的跨平台代理核心，支持多种代理协议和复杂的网络配置.

本仓库主要包括singbox示例配置以及快速,智能的部署脚本,可以一键部署以下协议并给出标准链接,具体实现见手动安装部分.

``wget https://raw.githubusercontent.com/Dichgrem/singbox-example/refs/heads/main/script/allinone.sh``,随后``bash ./allinone.sh``
